Getting started

APIdocs is an opinionated Gatsby.JS template for writing technical documentation. Clone, customize and start documenting.

Out-of-the-box, APIdocs comes with:

  • HTML-based content authoring,
  • responsive, mobile-friendly layout,
  • a dark and light color scheme,
  • Prism.js code highlighting,
  • Sublime-style client-side search,
  • near-100 Lighthouse score of the final static site.

To get started, follow the quick start tutorial. See editing navigation and further articles for more in-depth documentation. You can also check out gatsbyjs.org for some basics of Gatsby.
